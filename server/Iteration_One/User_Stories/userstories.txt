Ability to login, register if not logged in, and log out

Adi:

      secure.
    - As a user, I want to able to register on slackr with my first and last name, so that my account can be easily
      identified by other users whom I would want to message or recieve messages from.
      
    - As a slackr user, I want to be able to login with my email and password, as they are easy to remember.
    - As a slackr user, I want an error message to be displayed if incorret login details are provided, so I know
      that I am aware that I have entered wrong details.
    
    - As a slackr user, I want to be able to logout, so that no one else can access my account while I am away from

      to my email and have a means of reseting my password.
    - As a user, I want to be able to log out of Slackr, so that my account cannot be accessed once it has been logged out.
    - As a user, I want to be able to log into Slackr after logging out, so that I do not need to register a new account.
    - As a user, I want my account account preferences to be the same after logging into my account again, so that I do
      not need to repeatedly change my preferences.

Liam:
